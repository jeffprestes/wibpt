Sinopse

O advento do Bitcoin trouxe uma nova tecnologia: Blockchain. Porém Vitalik Buterin ampliou as suas possibilidades com os Smart Contracts quando criou o Ethereum. Com os Smart Contracts podemos criar aplicações que nunca sairão do ar e seus dados nunca serão perdidos. Usando uma linguagem simples, Solidity, a criação de Smart Contracts esta disponível para todos. Venha em nosso workshop se empoderar aprendendo essa revolucionária tecnologia.
